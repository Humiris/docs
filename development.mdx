Follow these steps to install and run Humiris on your operating system:

**Step 1**: Install Humiris:

<CodeGroup>
  ```bash npm
  npm install humiris-ai
  ```

```bash yarn
yarn add humiris-ai
```

</CodeGroup>

<Info>
  **Prerequisite**: At this step you will need your Moai Apikey to continue.{" "}
  <a href="https://docs.humiris.ai/quickstart">How to get my Moai Apikey</a> ?
</Info>

**Step 2**: Use a mixtuning via npm:

After creating your API key, you'll need to add your cloud provider API keys to our platform as indicated [here](https://docs.humiris.ai/quickstart).

Once you've done that, you'll be able to perform your mixtuning after completing the necessary action in the mixtuning sction on the platform.

```typescript Typescript exemple
import MoAi from "humiris-ai";

const apiKey = "your_moai_apikey";
const moai = new MoAi(apiKey);

const payload = {
  mixTuningId: "your_mix_tuning_id",
  prompt: "What is the weather today?",
  systemInstruction: "Please provide a concise answer.",
  mixInstruction: "Cost optimization.",
  temperature: 10,
  maxTokens: 100,
};

async function testUseMixtuning() {
  try {
    const response = await moai.useMixtuning(payload);
    console.log("Response:", response.data);
  } catch (error) {
    console.error("Error:", (error as Error)?.message ?? "Unknown error");
  }
}

testUseMixtuning();
```

**Step 3**: Use a mixtuning via api request:

```typescript api url
http://moai-service-app.humiris.ai/api/api-key-operators/use-mixtuning
```

```typescript api request
const response = await axios.post(this.apiUrl, payload, {
  headers: {
    "moai-api-key": this.apiKey,
    "Content-Type": "application/json",
  },
});
return response;
```

####Payload

```typescript api url
const payload = {
  mixTuningId: "your_mix_tuning_id",
  prompt: "What is the weather today?",
  systemInstruction: "Please provide a concise answer.",
  mixInstruction: "Cost optimization.",
  temperature: 10,
  maxTokens: 100,
};
```

- **mixTuningId**: you can find it on the platform, in your mixtuning list.

- **prompt**: add you prompt here.

- **systemInstruction**: add the system instruction here.

- **mixInstruction**: add your mix instruction here.

- **temperature**: must be an integer

- **maxTokens**: must be an integer
